<h3>Construtores</h3>
<p>Por padrão o Java tem o construtor padrão chamado de default.</p>
<p>Os construtores são métodos especiais usados para criar objetos da classe, com ele
é possível parametrizar o objeto criado e inicializar varáveis</p>
